# Tölvuleikur Prufa

## Markmið verkefnis
Þetta verkefni verður einfaldur vefjabyggður leikur þar sem lítill bolti á skjánum étur aðra minni boltana. Þegar boltinn vex verður hann hæfari til að éta stærri boltana, en hann getur aðeins étið þá sem eru minni en hann sjálfur.

## Leikjuhugmynd
- Spilarinn stjórnar einum bolta með mús eða bendill.
- Á skjánum birtast fleiri litlir boltar sem hægt er að eta.
- Þegar bolti étur annan bolta vex hann.
- Ef bolti er stærri en annar boltinn getur hann ét honum.
- Ef bolti er minni en annar boltinn getur hann ekki ét hann.
- Markmiðið er að vaxa og ná sem mestu stigi sem mögulegt er.

## Áætlun / agenda
1. Búa til grunninn í HTML, CSS og JavaScript.
2. Setja upp hreyfanlegan bolta sem spilarinn stjórnar.
3. Búa til nokkra litla bolta sem birtast á skjánum.
4. Láta bolta vaxa þegar hann étur minni bolta.
5. Gera regluna að aðeins minni boltar séu étanlegir.
6. Bæta við stigi, leikslokum og einföldu útliti.
7. Setja verkefnið undir Git og tengja við GitHub.

## Hugarfarslega uppbygging verkefnis
- index.html - aðalvefskjárinn
- style.css - útlit og hönnun
- game.js - leikjareglur og virkni
- assets/ - myndir, hljóð og önnur úrræði
- docs/ - skýrslur, áætlanir og athugasemdir

## Vinnuflæði
- Hugmyndir og skipulag í Obsidian
- Forritun og prófun í VS Code
- Útgáfa og rakning með Git/GitHub

## Miðlæg markmið

- Búa til einfaldan en skemmtilegan leik.
- Hafa verkefnið tilbúið til að deila og sýna öðrum.
